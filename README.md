# bash-tricks

| Command                   | Result                                             |
| :------------------------ | :------------------------------------------------- |
| **Ctrl + L**              | clear the actual window                            |
| `sudo !!`                 | run the last command with sudo privileges          |
| `grep -Ev '^#|^$' <file>` | display file content with comments and empty lines |
| **Crtl + R**              | reverse search in bash                             |
